**版本：** v1.0
**日期：** 2026-03-26
**面向读者：** App 使用者（家庭健康管理者、慢病患者及家属、Apple Watch 用户）

## 欢迎使用 AI Health Vault

AI Health Vault 是一款专为**家庭健康管理**设计的 iOS App。你可以用它记录全家人的体检报告、用药情况、就医经历和日常健康数据——所有数据保存在你的手机里，不上传任何服务器，你的健康隐私完全由你掌控。

**它能帮你做什么？**
- 一个 App 管理全家人的健康档案，不再在不同应用之间切换
- 体检报告不再只是压箱底的纸，随时查阅历史数据、掌握健康变化趋势
- 就医前快速回顾用药情况和既往病史，不再在诊室前慌乱翻找
- 连接 Apple Watch，自动同步步数、心率、睡眠等健康数据
- AI 用通俗语言解读体检指标，让健康数据真正"看得懂"
- 就诊前一键生成就诊摘要，高效利用有限的诊室时间

## 1. 快速开始

### 首次使用

1. 打开 App，按照引导设置**认证方式**（Face ID 或 PIN 码）
2. 点击右上角 **"+ 添加成员"**，为自己或家人创建第一份健康档案
3. 在**记录** Tab 开始录入体检报告、用药记录等健康数据
4. 进入**AI 助手** Tab，配置你的 Claude API Key，解锁 AI 健康分析功能

### 主界面导航

App 底部有四个主要入口：

| 图标 | 名称 | 用途 |
|------|------|------|
| 家庭 | 家庭成员 | 管理所有家庭成员档案，查看成员健康概览 |
| 记录 | 健康记录 | 录入和查看体检、用药、就医、日常等所有记录 |
| AI 助手 | AI 助手 | AI 健康解读、趋势分析、就诊准备、药物识别、每日健康计划 |
| 设置 | 设置 | 认证、家庭管理、App 信息 |

界面顶部的**成员选择器**让你一键切换到不同家庭成员，所有数据视图随之更新。

## 2. 保护你的健康数据 — 认证设置

健康数据是最私密的个人信息。每次打开 App，你需要通过身份验证才能访问数据。

### 设置认证方式

- **Face ID / Touch ID（推荐）：** 扫脸或指纹即可解锁，安全且便捷
- **6 位 PIN 码（备选）：** 若设备不支持生物认证，可设置 6 位数字 PIN

### 认证规则

- 生物认证失败 3 次后，自动切换为 PIN 码输入
- App 进入后台超过 **5 分钟**，重新打开时需再次验证
- 认证界面不显示任何健康数据，保障数据安全

> **提示：** 忘记 PIN 码？进入"设置 → 认证"，通过 Apple ID 验证身份后可重置。

## 3. 管理家庭成员

### 添加家庭成员

1. 进入**家庭** Tab，点击右上角 **"+"**
2. 填写成员信息：
   - **必填：** 姓名
   - **可选：** 生日、血型、性别、与你的关系（本人/配偶/父母/子女等）
   - 选择一个头像颜色或图标作为标识
3. 点击**保存**，档案创建完成

最多支持 **10 位**家庭成员。

### 切换当前成员

点击任意界面顶部的**成员头像**，从列表中选择要查看的家庭成员。所有记录和数据将自动切换为该成员的数据。

### 编辑与删除成员

- **编辑：** 进入成员详情页 → 点击右上角"编辑"
- **删除：** 编辑页面底部"删除成员"→ 确认（该成员的所有关联健康数据将一并删除，操作不可恢复）

## 4. 记录健康数据

进入**记录** Tab，点击右上角 **"+"** 选择记录类型开始录入。

### 体检报告

**什么时候用：** 每次体检后录入关键指标，建立个人健康数据库。

**怎么录入：**
1. 填写体检日期和机构名称
2. 点击"添加指标"，逐条录入体检项目（指标名称、数值、单位、参考范围）
3. 用颜色标记每项指标状态：绿色（正常）/ 黄色（偏高/偏低）/ 红色（异常）
4. 可添加备注（如医生建议、注意事项）

**拍照 + OCR 自动识别（推荐）：**
1. 创建体检记录时，点击"拍照添加"
2. 拍摄体检报告或从相册选择图片
3. App 使用 Apple Vision 框架自动识别报告中的指标数值（完全本地处理，无需网络）
4. 在可编辑列表中确认或修正识别结果后保存
5. 原始图片同时保存到记录中，可全屏查看

**使用建议：**
- 历年体检报告都录入后，同一指标的变化趋势将在趋势图表中清晰显示
- 录入后可点击"AI 解读"获取指标的通俗解释和健康建议

---

### 用药记录

**什么时候用：** 记录正在服用或曾经服用的药物，就医时一目了然提供用药史。

**录入内容：** 药品名称、剂量、服用频次、开始/结束时间、开药医生、适应症

**用药提醒：** 为当前用药设置提醒时段（早 08:00 / 中 12:00 / 晚 20:00），App 通过本地通知准时提醒，无需网络连接。

**使用技巧：**
- 记录列表分为**当前用药**和**历史用药**两个分组
- 药物停用时，将状态改为"已停用"而非删除，保留历史用药轨迹
- 就医前检查当前用药列表，告知医生避免药物冲突

---

### 就医记录

**什么时候用：** 每次看诊后记录关键信息，建立完整就医史，复诊时快速向医生提供背景。

**录入内容：** 就诊日期、医院、科室、医生姓名、主诉（这次为什么去看病）、诊断结论、处置建议、下次随访日期

**随访提醒：** 填写"下次随访日期"后可开启提醒，App 将在提前 1 天和当天 09:00 各发送一次本地通知。

**使用建议：**
- 就医后趁记忆新鲜，尽快录入诊断和建议，避免遗忘
- 就诊前可使用 AI 助手的"就诊准备"功能，自动整理症状和历史数据

---

### 日常追踪

**什么时候用：** 定期监测血压、血糖、体重等慢性指标，掌握长期变化趋势。

**支持追踪的数据类型：**

| 类型 | 单位 | 适用人群 |
|------|------|---------|
| 血压 | mmHg（收缩压/舒张压） | 高血压患者及家属 |
| 血糖 | mmol/L（空腹/餐后） | 糖尿病患者及家属 |
| 体重 | kg | 体重管理 |
| 心率 | bpm | 心脏健康关注者 |
| 体温 | °C | 发烧监测 |

**使用建议：**
- 每次测量后立即录入，数据越完整，趋势图越有参考价值
- 同类数据积累 7 天以上，可在"健康趋势"中查看变化曲线
- 授权 HealthKit 后，Apple Watch 数据将自动同步，无需手动录入

---

### 既往病史

**什么时候用：** 记录成员曾经确诊的疾病（如高血压、糖尿病史），作为健康背景信息的一部分。

既往病史、过敏史、慢性病/长期服药信息记录在**健康档案**的固定 Tab 中，作为每个成员的基础健康背景，随时可查阅和编辑。

## 5. 搜索与筛选

**全局搜索：** 在"记录" Tab 顶部输入关键词，可跨所有记录类型搜索——输入"血糖"找到所有相关记录，输入"北京协和"找到该医院的就诊记录。搜索结果实时更新，输入即搜索。

**按类型筛选：** 点击记录列表顶部的分类标签，快速切换到特定记录类型（体检 / 用药 / 就医 / 日常追踪）。

**按成员筛选：** 顶部成员选择器切换到对应成员，所有数据自动过滤为该成员的记录。

## 6. 连接 Apple Watch（HealthKit）

授权 AI Health Vault 读取 HealthKit 数据后，App 将自动同步你的 Apple Watch 健康数据：

| 数据类型 | 说明 |
|---------|------|
| 步数 | 每日步数汇总 |
| 心率 | 静息 / 运动心率记录 |
| 睡眠 | 睡眠时长和质量分析 |
| 体重 | Apple Watch 体重测量数据 |
| 血压 | 第三方血压设备同步数据 |
| 血氧 | Apple Watch 血氧饱和度 |

**如何开启：**
1. 进入"日常追踪"页面，首次进入会弹出 HealthKit 授权请求
2. 选择允许读取的数据类型（可单独授权）
3. 授权后 App 自动拉取最近 30 天历史数据，此后后台定期同步新数据

**关于数据来源标注：** HealthKit 自动同步的数据在列表中会显示"来自 Apple Health"标注，与手动录入的数据区分显示。

**选择不授权？** 完全没问题。App 的手动录入功能不受影响，你可以继续手动录入日常追踪数据。

## 7. AI 健康助手

AI 助手基于 Claude AI（claude-sonnet-4-6 模型），提供五大 AI 功能，帮助你真正理解自己的健康数据。

### 配置 API Key

首次使用 AI 功能需要配置你的 Claude API Key：
1. 进入"AI 助手" Tab，点击右上角设置图标
2. 输入你的 Claude API Key（由 Anthropic 提供）
3. Key 安全存储在 iOS Keychain 中，不会上传任何服务器

### 体检报告解读
录入体检报告后，点击"AI 解读"——AI 用通俗的语言告诉你：
- 哪些指标需要关注，为什么
- 整体健康状况评估
- 建议的后续行动

结果可保存到记录中，也可通过系统分享功能发送给家人。

### 趋势分析
进入"AI 助手 → 趋势分析"，选择健康指标和时间范围，查看带 AI 解读的趋势图表：血压是否在改善？血糖控制如何？图表配合解读，让数字变成行动建议。

### 就诊准备助手
就医前，在"AI 助手 → 就诊准备"中：
1. 输入本次就诊原因/症状
2. AI 自动整理你的症状和历史数据，生成就诊摘要
3. 复制或分享摘要，在有限的诊室时间里高效向医生说清楚

### 药物识别
在"AI 助手 → 药物识别"中，输入药品名，AI 返回药品用途、注意事项及常见相互作用提示，帮助你做更科学的用药决策。

### 每日健康计划
在"AI 助手 → 每日健康计划"中，AI 基于你的综合健康数据（体检记录、日常追踪、用药情况）生成个性化每日健康建议，包括饮食、运动、用药提醒等。

### 医学术语查询
查看体检报告时，遇到不懂的专业术语可直接点击，App 会调用 AI 给出通俗解释，无需离开 App 去搜索。

### 重要说明
- 调用 AI 分析时，App 仅向 AI 传送必要的结构化指标数据（指标名称 + 数值），**不传照片，不传姓名等身份信息**
- 每次使用 AI 功能前，App 会告知你哪些数据将被使用，由你确认
- AI 分析需要网络连接；无网络时，所有本地记录功能正常使用，不受影响
- AI 仅供参考，不能替代专业医疗建议

## 8. 健康趋势图表

在"AI 助手 → 趋势分析"或直接从日常追踪记录进入，可查看健康指标的变化趋势：

- **异常点标注**：超出正常范围的数据点用颜色高亮
- **跨时间对比**：体检记录中同类指标可跨年度对比

## 10. 隐私与数据安全

**数据存储在哪里？**
你的所有健康数据保存在**手机本地**，默认不上传任何云端服务器。

**数据加密了吗？**
是的。AI Health Vault 使用 iOS 内置的文件系统加密（Data Protection），数据以加密形式存储，即使设备丢失也无法被他人读取。

**AI 分析会上传我的数据吗？**
仅在你主动点击"AI 解读"时，App 才会将精简后的**指标数据**（不含照片和个人身份信息）发送给 Claude AI 服务。每次调用前 App 会明确告知。

**API Key 安全吗？**
Claude API Key 存储在 iOS Keychain 中（系统级安全存储），不会保存在 App 数据库或传输到任何第三方服务器。

**iCloud 同步安全吗？**
iCloud 同步功能（v1.1 规划）**默认关闭**，需要你在设置中主动开启。开启后数据通过苹果 CloudKit 加密传输，关闭同步不会删除本地数据。

**App 收集使用数据吗？**
不收集任何使用数据、行为数据或崩溃报告（不集成第三方数据分析 SDK）。

## 11. 常见问题

**Q: App 最多支持多少家庭成员？**
A: 最多 10 位家庭成员。

**Q: 删除家庭成员后，数据能找回来吗？**
A: 不能。删除成员时会弹出二次确认提示，确认后该成员的所有关联数据永久删除。请谨慎操作。

**Q: 手机丢了，数据还在吗？**
A: 数据仅存储在设备本地，如未开启 iCloud 同步（v1.1 规划），更换手机后数据无法恢复。

**Q: 需要注册账号吗？**
A: 不需要。AI Health Vault 没有账号系统，应用和数据完全在本地运行。

**Q: AI 解读是免费的吗？**
A: AI 功能需要使用你自己的 Claude API Key（由 Anthropic 提供，按使用量计费）。进入"AI 助手 → 设置"配置 Key 后即可使用。

**Q: OCR 识别需要网络吗？**
A: 不需要。OCR 使用 Apple Vision 框架在本地处理，完全离线可用。

**Q: 支持 iPad 吗？**
A: 当前版本针对 iPhone 优化（竖屏布局）。iPad 支持在后续版本规划中。

**Q: 最低支持哪个 iOS 版本？**
A: iOS 17.0 及以上（要求 iPhone 12 及更新机型）。

**Q: 体检报告图片会占用多少存储空间？**
A: 图片以高质量格式存储在本地，建议为每条体检记录保留 3-5 张图片，避免占用过多存储空间。

**Q: 用药提醒准时吗？**
A: 用药提醒使用本地推送通知，无需网络即可收到。提醒时段为早（08:00）/ 中（12:00）/ 晚（20:00），可在用药记录中按需开启。
